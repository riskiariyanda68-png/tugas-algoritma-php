## Judul Program
**Program Menghitung Total Pembayaran dan Diskon**

---

## Deskripsi
Program ini digunakan untuk menghitung total pembayaran belanja dengan sistem diskon bertingkat. Program akan menghitung diskon berdasarkan total belanja, status keanggotaan (member), dan hari spesial. Semua hasil perhitungan ditampilkan dalam format Rupiah dengan pemisah ribuan (titik) untuk memudahkan pembacaan.

**Fungsi utama program:**
- Menghitung diskon otomatis berdasarkan total belanja
- Memberikan diskon tambahan untuk member (Silver, Gold, Platinum)
- Memberikan diskon tambahan pada hari spesial
- Menampilkan rincian diskon yang didapat
- Menyimpan riwayat transaksi

---

## Fitur
- **Perhitungan Diskon Bertingkat** - Diskon berdasarkan total belanja (5% - 20%)
- **3 Jenis Member** - Non Member, Silver (5%), Gold (10%), Platinum (15%)
- **Diskon Hari Spesial** - Tambahan diskon 10% pada hari tertentu
- **Rincian Diskon** - Menampilkan detail setiap diskon yang didapat
- **Format Rupiah** - Menampilkan angka dengan format Rp 1.000.000 (titik ribuan)
- **Input Otomatis** - Format Rupiah otomatis saat mengetik angka
- **Riwayat Transaksi** - Menyimpan 10 transaksi terakhir
- **Responsive Design** - Tampilan menyesuaikan di berbagai ukuran layar
